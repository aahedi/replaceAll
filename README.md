# replaceAll
replaceAll library js by [Aahedi](https://github.com/aahedi/)
## Usage

```
replaceAll(
'123hedi456 abchediefg 123hedi abchedi hedi456 hediefg 123Hedi456',
'hedi',
'****', 
{prefix_is:'123',
suffix_is:'456',
prefix_not:'123',
suffix_not:'456',
insensitive:true})
```

## result...

```
123hedi456 abc****efg 123hedi abc**** hedi456 ****efg 123Hedi456
```
